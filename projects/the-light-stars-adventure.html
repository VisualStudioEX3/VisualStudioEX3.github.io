The Light Star's Adventure

A point & click adventure made in Visual Basic 6.0.

Source Code The Light Star's Adventure (1999 - 2000) on GitHub

My first game project as programmer, a Point & Click Graphic Adventure in first person view, placed on a spaceship invaded by an enemy army. The player taking the role of a prisioner who tries to escape from the spaceship in middle a battle between 2 military factions, walking through corridors, interacting with elements like wall panels, computers, elevators, characters with complex dialogues, and even shooting enemies in certain scenes.

The game only contain an unfinished level, playable from start to end, with several scenes (15 rooms aprox.) and puzzles to solve in the path, and count with some checkpoints to ease play the level if the player dead in any circunstance.

The graphics are handmade, using the mouse, in Microsoft Paint. All graphics are statics, without animations or transitions. The game not contain any audio. Was developed in Visual Basic 6.0 entirely using Windows Forms controls: PictureBox, Image, Label, Timer and Progress Bar controls. If you want to know more about how the game is technically developed, you can found more information in the GitHub readme repository.
