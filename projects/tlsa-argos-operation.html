T.L.S.A.: Argos Operation

Prototype of the second version of the The Light Star's Adventure point & click game.

Genre 2D, Adventure, Point & Click
Source Code T.L.S.A.: Argos Operation (2001) on GitHub

A year after The Light Star Adventure project I decided to start a new version but implementing new ideas, like the capacity to create scrollable scenes, to ease create more rich and complex scenarios in the same place without the needed to implementing different views (except for backwards or zooms like in the original game).

Most of the focus in this prototype was the visual aesthetics. I improved the drawing of the backgrounds and characters and also improved the GUI using a custom style instead of raw-default Windows style, using custom icons and cursors, and moved all texts and dialogues to the bottom of the screen.

The prototype shows a scene placed on the roof of a futuristic building on an unknown planet. The player is currently under arrest by two imperial soldiers. One of them is aiming you with his shootgun. After a short dialogue with him the soldier starts to shoot you. You can shoot the soldiers with your pistol and kill them, after that you can scroll the view to search items, like a health pack to recover life.

One difference between this game and the first is the implementation of life points to characters and player, and also ammo for player weapons. Another new feature is the audio, the weapons, interactions and characters can play sound effects.

Like the first one, this game is developed using Visual Basic 6.0 and Windows Forms controls, practically developed from scratch, no reusing code from the first game, and used Microsoft Paint for create the graphics.
