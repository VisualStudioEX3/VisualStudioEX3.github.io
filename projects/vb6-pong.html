Mi first try to develop a game, a version of classic Atari Pong.

The game aesthetics want to recreate the original monochorme visuals of the Atari Pong from the 70s. The graphics are composed using Windows Controls like Labels and Shapes controls. The game doesnâ€™t had audio implemented.
